Alter – Android Social Network for Habit Tracking and Addiction Recovery ( pictures in the repport file)

Alter (New Step) is a cloud native, partly serverless and architecture hybride android application developed in Java with Firebase and SQLite, combining habit tracking and social networking to help users overcome bad habits and build healthier routines through community support.

Project Overview

This application was developed as part of a university project by a team of six computer engineering students at Badji Mokhtar University – Annaba.

Alter allows users to:

Track their habits and daily progress,

Share experiences and encouragement through community posts,

Chat privately or within support groups,

Receive reminders and motivational messages to stay consistent.

🚀 Main Features
🩵 Habit Tracking

Add, edit, or delete habits

Daily journaling with mood and notes

Progress streak tracking

Local offline tracking using SQLite

🌐 Social Features

Create and interact with posts (likes, comments, reports)

Join or create support groups

Private and group messaging

User profiles with privacy settings

Real-time synchronization with Firebase

🧩 Administrator Panel

Manage reported posts and users

Ban/unban users or restrict comments/posts

Moderate content for a safe community space

🤖 Milo – AI Assistant

Built-in assistant to guide users and answer basic questions about using the app

⚙️ Technologies Used

Programming Language: Java

IDE: Android Studio

Online Database: Firebase Firestore

Offline Database: SQLite

Authentication & Storage: Firebase Auth + Firebase Storage

cloud service : firebase

Architecture: MVC

Version Control: Git / GitHub (branches, pull requests, merges)
